# 就活管理ツール

企業の選考状況（未応募・ES・面接・内定・落選）を一覧管理するHTMLツールです。Claude APIで企業情報を自動取得できます。

## 機能

- 企業名・業界・選考状況・メモの登録・編集・削除
- 検索・選考状況フィルター
- Claude APIによる企業概要・事業内容・就活の特徴の自動取得
- データはブラウザの localStorage に保存

## 使い方

### 1. プロキシを起動（AI機能を使う場合）

```bash
node 就活管理-proxy.js
```

### 2. HTMLを開く

`就活管理.html` をブラウザで開きます。

### 3. APIキーを設定

画面右上の「API設定」から [Anthropic](https://console.anthropic.com/) のAPIキーを入力してください。

## ファイル構成

| ファイル | 説明 |
|---------|------|
| `就活管理.html` | メインアプリ |
| `就活管理-proxy.js` | Claude API用ローカルプロキシ（ポート 3456） |

## 注意

- APIキーはソースコードに含めず、ブラウザの設定画面から入力してください
- プロキシはローカル開発用です。本番公開時はサーバー側でAPIを呼び出す構成を推奨します

## 必要環境

- モダンブラウザ
- Node.js（AI機能・プロキシ利用時）
